# 三体舰队运行监控 (Three Body Fleet Monitor)

一个基于《三体》世界观设计的舰队航行实时监控终端。采用科幻感十足的仪表盘界面，展示三体第一舰队向太阳系进发过程中的核心数据。

## 项目特点
- **实时数据模拟**：基于物理公式模拟舰队在加速、滑行、减速阶段的速度与距离变化。
- **纪元同步系统**：自动识别并展示当前年份所属的三体纪元（如危机纪元、威慑纪元等）。
- **时空穿梭 (Warp Mode)**：支持自由跳转到 1982-2400+ 年间的任意时刻，观测历史或未来的舰队状态。
- **背景自动切换**：支持多张背景图轮询切换（默认每 3 分钟），并伴有平滑的过渡效果。
- **科幻视觉效果**：包含动态星空背景、CRT 扫描线、发光仪表盘等视觉元素。
